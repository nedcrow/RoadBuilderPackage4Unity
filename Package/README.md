# Road Builder Package

Unity에서 도로를 만들 수 있는 클릭 기반 시스템입니다.

## Features

- 클릭 기반 도로 생성 시스템
- 자동 스냅 기능 (기존 도로에 연결)
- 베지에 곡선 지원
- 직선/곡선 모드 전환
- 도로 청크 삭제 기능
- New Input System 및 Legacy Input 지원
- 지형 추적 (Terrain Following) - 도로가 지형 높낮이를 따라감
- Shader Graph 기반 커스텀 도로 셰이더

## Installation

### Via Git URL

1. Unity Package Manager를 엽니다 (Window > Package Manager)
2. "+" 버튼을 클릭하고 "Add package from git URL"을 선택합니다
3. 다음 URL을 입력합니다:
   ```
   https://github.com/[YOUR_USERNAME]/RoadBuilderPackage4Unity.git?path=/Package
   ```

### Import Sample

1. Package Manager에서 RoadBuilderPackage4Unity를 선택합니다
2. Samples 탭에서 "Example Scene"을 찾습니다
3. Import 버튼을 클릭합니다

샘플이 다음 경로에 임포트됩니다:
```
Assets/Samples/RoadBuilderPackage4Unity/1.0.0/Example Scene/
```

## Quick Start

1. 샘플을 임포트합니다
2. Example Scene을 엽니다
3. Play 모드로 진입합니다
4. 좌클릭으로 도로를 생성합니다

## Requirements

- Unity 2021.3 or later
- Universal Render Pipeline (URP)
- Input System Package (optional, but recommended)
